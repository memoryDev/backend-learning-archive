# 📚 Backend Learning Archive

백엔드 개발자로 성장하기 위해 학습한 기초 문법, 프레임워크 핵심 원리, 그리고 다양한 기술 스택의 실습 코드를 기록하는 저장소입니다. 
지속적인 학습과 코드 리뷰를 통해 더 나은 설계를 고민합니다.

---

## 📂 카테고리별 학습 내용

### 🌱 01. Java (언어 기초)
- **자바 중급/고급 문법**: 예외 처리, 제네릭, 컬렉션 프레임워크 학습
- **객체 지향 프로그래밍**: 다형성, 추상화, 캡슐화 등 OOP 핵심 원리 적용 실습

### 🍃 02. Spring MVC (웹 개발 핵심)
- **Spring Core**: 제어의 역전(IoC), 의존관계 주입(DI) 등 스프링 컨테이너 이해
- **Spring MVC**: 서블릿부터 핸들러 어댑터, 뷰 리졸버까지 웹 요청 처리 흐름 학습
- **Validation & Login**: 데이터 검증, 세션 및 쿠키를 활용한 로그인 처리 실습

### 💾 03. Database & JPA (데이터 관리)
- **JDBC/DB1**: 데이터베이스 연결 기초 및 트랜잭션 개념 학습
- **JPA 기초**: 영속성 컨텍스트, 엔티티 매핑, 연관관계 관리
- **Spring Data JPA & QueryDSL**: 선언적 쿼리 사용 및 복잡한 동적 쿼리 최적화 실습

### 🚀 04. Spring Advanced (고급 및 운영)
- **Spring Advanced**: 디자인 패턴(프록시, 템플릿 메서드 등)을 활용한 부가 기능 구현
- **Security & JWT**: OAuth2, JWT를 활용한 인증 및 인가 시스템 구축 학습
- **Actuator**: 스프링 부트 애플리케이션의 모니터링 및 관리 지표 학습

### 🛠️ 05. Etc Study (기타 학습)
- **Frontend 기초**: React(한입 리액트) 등 프론트엔드 기본 문법 실습
- **기타 실습**: 학습 목적의 간단한 토이 프로젝트 코드 보관
